Lab 2: Robot Obstacle Avoidance Using Ultrasonic Sensors

Experiment Overview

This experiment focuses on realizing the robot's obstacle avoidance function through ultrasonic sensors. The robot will detect the distance of obstacles in front in real time, and automatically adjust the movement direction to avoid collisions when the distance is less than the safe threshold.

Key experimental steps:

  1. Calibrate the ultrasonic sensor of the robot
  2. Write code to read sensor distance data
  3. Implement obstacle avoidance logic based on distance data
  4. Test the obstacle avoidance effect in a complex environment

üìù Experimental Code

# Python code for robot obstacle avoidance with ultrasonic sensor
import robot_control_lib as rcl
import time

# Initialize robot and ultrasonic sensor
robot = rcl.Robot()
ultrasonic_sensor = rcl.UltrasonicSensor()

# Safe distance threshold (cm)
SAFE_DISTANCE = 30

def obstacle_avoidance_demo():
    """Demo of robot's obstacle avoidance function"""
    print("Obstacle avoidance system started...")
    while True:
        # Read current obstacle distance
        distance = ultrasonic_sensor.get_distance()
        print(f"Current obstacle distance: {distance} cm")
        
        if distance < SAFE_DISTANCE:
            # Obstacle detected, perform avoidance action
            print("Obstacle detected! Starting avoidance...")
            robot.stop()
            time.sleep(0.5)
            robot.turn_right(duration=2)
            robot.move_forward(duration=3)
        else:
            # No obstacle, move forward normally
            robot.move_forward(duration=1)
        
        # Add a small delay to avoid data reading too fast
        time.sleep(0.1)

# Call the demo function (press Ctrl+C to stop)
if __name__ == "__main__":
    try:
        obstacle_avoidance_demo()
    except KeyboardInterrupt:
        print("Experiment stopped manually.")
        robot.stop()

Lab 3: Robot Path Planning Based on Vision

Experiment Overview

This experiment realizes the robot's autonomous path planning function through the vision module. The robot collects environmental images through the camera, identifies the target path and obstacles, and plans the optimal moving route to reach the designated target point.

Key experimental steps:

  1. Initialize the robot vision module and calibrate the camera
  2. Implement image processing and path recognition algorithms
  3. Write path planning and movement control code
  4. Verify the path planning effect in the preset scene

üìù Experimental Code

# Python code for robot vision-based path planning
import robot_control_lib as rcl
import cv2
import numpy as np

# Initialize robot and vision module
robot = rcl.Robot()
vision_module = rcl.VisionModule()

def path_planning_demo():
    """Demo of robot's vision-based path planning function"""
    print("Vision-based path planning system started...")
    
    # Capture environmental image
    frame = vision_module.capture_image()
    if frame is None:
        print("Failed to capture image, experiment aborted.")
        return
    
    # Process image and recognize path
    processed_frame = vision_module.process_image(frame)
    path_points = vision_module.recognize_path(processed_frame)
    
    if len(path_points) == 0:
        print("No valid path detected.")
        return
    
    print(f"Valid path detected, {len(path_points)} waypoints found.")
    
    # Follow the planned path to move
    for point in path_points:
        print(f"Moving to waypoint: ({point[0]}, {point[1]})")
        robot.move_to_target(point[0], point[1])
        time.sleep(1)
    
    print("Path planning completed, robot reached the target.")
    robot.stop()

# Call the demo function
if __name__ == "__main__":
    path_planning_demo()
