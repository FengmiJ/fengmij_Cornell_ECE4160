Task4 Notification Handler

For this task, I add an additional function in jupyter lab called time_handler. It first converts the binary date into a string and then split it by finding ‚ÄúT:‚Äù. The example shown here indicates how it is performed in in the previous GET_TIME_MILLIS, and as we can see, it can print the time without the print command.


        def time_handler(uuid,byte_array):
            string_value = byte_array.decode("utf-8")
            print("Received string:", string_value)
            if "T:" in string_value:
                time_millis = string_value.split(":")[1]
                print(f"Extracted Time: {time_millis} ms")
        ble.start_notify(ble.uuid['RX_STRING'], time_handler)
        ble.send_command(CMD.GET_TIME_MILLIS, "")
        
# Python code for basic robot movement control
# Import the robot control library (simulated library for demonstration)
import robot_control_lib as rcl

# Initialize the robot
robot = rcl.Robot()

def basic_movement_demo():
    """Demo of robot's basic movement functions"""
    print("Robot starts moving forward...")
    robot.move_forward(duration=3)  # Move forward for 3 seconds
    
    print("Robot starts turning left...")
    robot.turn_left(duration=1)     # Turn left for 1 second
    
    print("Robot starts moving backward...")
    robot.move_backward(duration=2) # Move backward for 2 seconds
    
    print("Robot starts turning right...")
    robot.turn_right(duration=1)    # Turn right for 1 second
    
    print("Experiment completed, robot stops moving.")
    robot.stop()

# Call the demo function
if __name__ == "__main__":
    basic_movement_demo()
